## Форк Telegram с поддержкой XRay (ru)

В связи с тем что в России начали замедлять Telegram, многие люди стали пользоваться прокси. Но реализация MTProxy не позволяет ему быть таким же устойчивым к блокировкам как XRay, да и прокси серверов намного меньше чем VPNов. Поэтому я решил сделать форк Telegram с поддержкой VLESS.

# Что реализовано
1. Добавление конфигов из буфера обмена
2. Поддержка подписок (в том числе с HWID)
3. Авто тестинг серверов

Надеюсь что этот форк кому то пригодится.




## Telegram fork with XRay support (en)

Due to the fact that Telegram has been banned in Russia, many people have started using proxies. However, the implementation of MTProxy does not allow it to be as resistant to blocking as XRay, and there are far fewer proxy servers than VPNs. Therefore, I decided to make a Telegram fork with VLESS support.

# What has been implemented
1. Adding configurations from the clipboard
2. Subscription support (including HWID support)
3. Automatic server testing

I hope this fork will be useful to someone.
